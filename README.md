# AIRBNB_DASHBOARD-POWERBI
An interactive Power BI dashboard built using Airbnb listing data to analyze property performance, pricing, host activity, neighbourhood trends, room types, and customer ratings. The project includes KPIs, DAX measures, and visual insights to support data-driven decisions.

# 🏡 Airbnb Analytics Dashboard (Power BI)

## 📖 Project Overview

This project presents an interactive Power BI dashboard developed using Airbnb listing data. The dashboard provides comprehensive insights into property listings, pricing, availability, room types, host performance, customer ratings, and neighbourhood trends.

The goal of this project is to demonstrate business intelligence skills by transforming raw Airbnb data into meaningful visualizations and KPIs that support strategic decision-making.

---

# 🎯 Objectives

- Analyze Airbnb listing performance.
- Compare room types across neighbourhoods.
- Monitor pricing trends.
- Evaluate host activity.
- Identify high-performing locations.
- Build an interactive dashboard using Power BI.
- Create business-ready insights using DAX measures.
